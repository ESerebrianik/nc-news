## About the Project

NC News is a full-stack news application built with React and a RESTful API.

Users can:

- View a list of articles
- Filter articles by topic
- View articles by author
- Sort articles by date, votes, or comment count
- Like articles
- View and post comments
- Delete their own comments

The app communicates with a custom-built backend API.

## Running the Project Locally

1. Clone the repository:
   git clone https://github.com/ESerebrianik/nc-news.git

2. Navigate into the project folder:
   cd nc-news

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open the local development URL shown in the terminal (usually http://localhost:5173)
